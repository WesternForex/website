Sections

Working with PHP Structure

The Theme presents many sections inside

Sections are divided in php files and they located in ../sections/

You can include these in your index.php to create your home page or in other page:

  • Blog - Include th list of articles
  • Callout - Write an highlited message with title, subtitles and icons
  • Clients - Add a carousel of logos
  • Contact - Add a form contact and one box address
  • Counters - Add some services counter
  • Features - Add box for your info
  • Form - Add another form contact
  • Images - Add some animated images
  • Map - Add a google map
  • Overwiev - Add box details
  • Playground - Add a video section
  • Portfolio - Add a grid list of projects
  • Quotes - Set multiple message by some people
  • Services - Add others box informations
  • Team - Add box for your team
  • Widgets - Add an area widgets

To use section in your page or index, include related php file with php include function
e.g. include('blog.php');

Working with HTML Structure

The sections are entirely splitted in a single page.
You can recognize a section by its id.
E.g. section blog has id="blog" and class="section"

Note! - To make easy the page composition, we have also cloned the sections in html files in order to copy and past the content in other page.
Find out the sections in the folder sections.

Custom Colors

PHP Structure

In the theme are included 2 variables related to 2 principals colors of the theme.
These are declared in config.php

You can edit $fs_clr and $nd_clr and set them with hex or rgba color code

In adding you can edit also the alpha color, these are used for parallax layer or bg transparent - $fs_alpha, $nd_alpha

In head.php are inclused other theme colors, these are fixed by the develop team, but if you want, you could change after.

These colors are white .wh-clr, clear grey .gr1-clr, medium grey .gr2-clr and dark grey .gr3-clr. These are used for title, subtitle, header section, and various box styles.

In addition to using class to set font color, you can use the color classes to set border-color (e.g. .brd-gr2-clr), background (e.g. .bg-fs-clr), and hover color (e.g. .nd-clr-hov)

Below the custom colors in head.php, are present others customizations releated to some classes of the theme. Edit with care!

GRID SYSTEM

The theme included a responsive grid based on flow columns

The classes are located in grid.css

You can use these classes to insert lined up content in your page

  • .col-1-6 - Attach a width of 1/6 of the content
  • .col-1-5 - Attach a width of 1/5 of the content
  • .col-1-4 - Attach a width of 1/4 of the content
  • .col-1-3 - Attach a width of 1/3 of the content
  • .col-1-2 - Attach a width of 1/2 of the content

In adding to fit better your page are present also other grid classes as .col-2-3, .col-2-4, .col-3-4, .col-2-5, .col-3-5, .col-4-5, .col-2-6, .col-3-6, .col-4-6, .col-5-6.

SPACINGS

You can use useful classes related to padding and margin to edit your content.

These classes are located in custom.css

  • .marg-25 - attach 25px of margin to tag
  • .marg-50 - attach 50px of margin to tag
  • .marg-75 - attach 75px of margin to tag
  • .marg-100 - attach 100px of margin to tag
  • .padd-25 - attach 25px of padding to tag
  • .padd-50 - attach 50px of padding to tag
  • .padd-75 - attach 75px of padding to tag
  • .padd-100 - attach 100px of padding to tag

Other possible combinations - You can mix direction or axis y/x

  • .padd-top-25 - attach 25px of padding-top to tag
  • .padd-bott-50 - attach 50px of padding-bottom to tag
  • .margin-left-75 - attach 75px of margin-left to tag
  • .margin-right-100 - attach 100px of margin-right to tag
  • .padd-x-25 - attach 25px of padding left and right to tag
  • .padd-y-50 - attach 50px of padding top and bottom to tag
  • .marg-x-75 - attach 75px of margin left and right to tag
  • .marg-y-100 - attach 100px of margin top and bottom to tag

TRANSITIONS

You can use useful classes to attach some css3 animations to your box content

The classes are located in custom.css

  • .transit - add a transition about 0.3s erase-in-out
  • .transit-fade - add a fade transition
  • .transit-left - add a transition from left
  • .transit-right - add a transition from right
  • .transit-top - add a transition from top
  • .transit-bottom - add a transition from bottom
  • .transit-bouncein - add a bounce in transition
  • .transit-flip - add a 3d transition
  • .transit-rotate - add a rotate transition
  • .transit-pulse - add a pulse transition
  • .transit-bounce - add a bounce transition
  • .transit-words - add a transition to the letters of a text

In adding to customize your transitions, you can set some parameters inserting these data attribute in your html tag (not valid for transit-words)

  • data-delay="millisec" - define the duration of the transition
  • data-appear="true/false" - set the start of transition when scroll page

VIDEO

Video Section can be set with some parameters

  • To create a video section, you must to create a container div before and mark it with an 'id'.
  • Then You need to include inside an 'anchor' tag.
    At this 'a' tag you need to attach a data attribute (data-property) with the following properties : videoURL (link video), containment ('id' of the wrapper'), showControls (true/false), autoPlay (true/false), mute (true/false), startAt (seconds from start video) and opacity (from 0 to 1).
  • In the end you have to attach a class and an id to 'a' tag

Check the content located in playground file section to start to edit it and using it

LIGHTBOX

To set a lightbox image, you must attach the class .lightbox to the 'anchor' which contains the href="url" related to the image source.

To set a lightbox group, you must also attach the rel="group" attribute to the 'anchor'.

CHANGE PAGE

To avoid to do appear the curtain white after page change, you have to attach a data-curtain="false" attribute to the 'anchor' tag.
